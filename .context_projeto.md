# Contexto do Projeto: Calculadora Web Gemini

## Objetivo

Criar uma calculadora funcional em Python com interface Streamlit.

## Regras de Arquitetura

- Separar lógica matemática da interface visual.
- Usar tipos de dados flutuantes (float) para cálculos.
- Tratar erro de divisão por zero com mensagens amigáveis.

## Estado Atual

1. [Done] Definição da lógica em engine_calculo.py
2. [Done] Criação da interface em app_interface.py
3. [Aguardando] Teste local e Deploy.

## Stack Técnica

- Python 3.10+
- Streamlit (Frontend)
- MCP Context7 para persistência de estado.
